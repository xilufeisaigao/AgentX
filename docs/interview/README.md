# AgentX Interview Bank

这个目录专门存放 AgentX 项目的面试题归档，不和运行真相文档混写。

使用原则：

1. 面试题优先按主题归档，不按会话顺序堆叠。
2. 每个问题单独成段，不做题号，方便后续插入新问题。
3. 每个问题都要标 `重要程度：高 / 中 / 低`。
4. 新问题优先插入已有主题文档；如果没有合适主题，再在本目录下新建文档。
5. 答案优先保留“项目当前真相”和“目标方案”之间的边界，避免面试口径和代码真相混淆。
