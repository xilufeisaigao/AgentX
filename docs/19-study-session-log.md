# AgentX 每日学习复盘日志

这份文档只在用户说 `结束学习` 时更新。

用途：

1. 记录今天实际学了什么。
2. 提炼今天最重要的概念。
3. 保存今天反复追问的点。
4. 留下面试题和复习入口。

## 更新规则

每次结束学习时，agent 追加一个新日期小节，格式保持统一：

1. 今日学习主题
2. 今日实际讲到的代码位置
3. 今日重要概念
4. 今日反复提问
5. 今日面试题
6. 下次从哪里继续
